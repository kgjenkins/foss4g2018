Using Deep Learning to Derive 3D Cities from Satellite Imagery

Eric Culbertson (Astraea, Inc.)

Detection and reconstruction of 3D buildings in urban areas has been a hot topic of research due to its many applications, including 3D population density studies, emergency planning, and building value estimation. Standard approaches to extract building footprint and measure building height rely on either aerial or space borne point cloud data, which in many areas is unavailable. In contrast, high resolution satellite imagery has become more readily available in recent years, and could provide enough information to estimate a building’s height.

Cloud Optimized GeoTiffs: enabling efficient cloud workflows

Eugene Cheipesh (Azavea, Inc.)

The Cloud Optimized GeoTIFF (COG) is a key component to enabling cloud-native geospatial workflows. An important benefit is the ability for the client to stream just the portion of data that is needed through the use of HTTP GET byte-range requests. COGs enable faster reading, writing, and processing of data on the cloud without the need for local copies.

The Map Algebra Modeling Language (MAML), originally developed at Azavea to expose complex raster processing through a graphical interface, allows clients to craft programs through the direct construction and manipulation of map algebra expression trees.

GeoTIFF.io: Pain-Free Raster Analysis

Daniel Dufour (GeoTIFF.io)

GeoTIFF.io is an open-source website that makes viewing and analyzing rasters like satellite imagery easier than ever before.  GeoTIFF.io processes satellite imagery, biomass grids, public health data, population rasters, and many other types of data.  It runs completely inside of your web browser with no server calls, so you can use it on your desktop or your smart phone and in a disconnected environment.  This presentation will go over several use cases and how to use the basic statistical tools like sum and histograms.

CERES Maps: an open platform for open research geodata constructed with FOSS.

Todd Slind (SpatialDev)

An initial building block in CGIAR’s Big Data Initiative, CERES Map is a raster data catalog and analysis tool designed for geoscientists needing to publish their geospatial research and for the discovery of and access to these outputs. The technical architecture includes several FOSS components and is comprised of four main modules: 1) Metadata database; 2) API; 3) Catalog Application; and 4) Map Viewer application. The presentation will cover an overview of the use cases and features of CERES Map.

Simplify your Mapping Applications with EMP

Jenifer Cochran

Learn how to create consistent mapping applications with ease on the Extensible Mapping Platform(EMP). EMP is an open-source API available for both Web and Android that allows you to choose your mapping engine and get started with common geospatial functions. The API allows you to write map applications quickly as well as have different mapping engines available to you without having to change your code. Stop re-writing the same boiler-plate code for every map you create and use EMP.

Take-out and the data problem: Human Geography and OSM

James Rummel (Reinventing Geospatial Inc)
Steven Lander (Reinventing Geospatial, Inc.)

Mapping the world is hard; mapping the groups and movements of human beings can be even harder. Human Geography is the study of these movements and groups, and like many other geospatial fields it suffers from a lack of good, up-to-date data. Open Street Map (OSM) data provides a feature rich data source of the Earth and is contributed to voluntarily and regularly by the public. OSM includes a wide variety of tags and among other things can be used to identify what a particular piece of land is being used for and by whom.

mago3D. Let's integrate BIM/AEC with 3D GIS on your web browser.

jinwoo PARK (Gaia3D Inc,)
Byeongcheol Bae (Gaia3D)

Although there have been numerous attempts to integrate BIM and 3D GIS on a single web platform, the outcome of those attempts are not so satisfactory to date. The difference of data models, the massive number of objects to be rendered, the big volume of data files, and proprietary technologies are among the major barriers that hindered seamless integration of BIM and 3D GIS on a web environment. This session introduces 'mago3D', an open source based GeoBIM platform that can manage, handle, and visualize massive 3D data from BIM and 3D GIS simultaneously on a web browser.

Daniel Carver (NASA DEVELOP - SSAI)

Raster Vision: Deep Learning for Aerial and Satellite Imagery

Lewis Fishgold (Azavea)

Increasingly, deep learning is being used to make sense of the vast quantity of satellite and aerial imagery being generated each day. Although there are many open source libraries for deep learning, they are unable to handle geospatial datasets without the addition of a non-trivial amount of extra data processing glue code. Over the past year, we have prototyped workflows for the computer vision tasks of classification, object detection, and semantic segmentation.
